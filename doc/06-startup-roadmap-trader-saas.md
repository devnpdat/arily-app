# 06. Startup Roadmap – Trader SaaS (12 Months Plan)

Tài liệu này mô tả **lộ trình phát triển startup SaaS quản lý thương lái nông sản trong 12 tháng**.

Mục tiêu:

- Xây dựng **MVP trong 3 tháng**
- Có **100–300 thương lái sử dụng thật**
- Tạo **doanh thu SaaS ổn định**
- Chuẩn bị **scale toàn quốc**

---

# 1. Tầm nhìn sản phẩm

Xây dựng nền tảng:

**“Operating System cho thương lái nông sản”**

Quản lý:

- thu mua
- công nợ
- kho
- bán hàng
- logistics
- nhân công
- phân tích giá

---

# 2. Quy mô thị trường

Việt Nam ước tính:

- >300,000 thương lái nhỏ
- >50,000 thương lái vừa
- >10,000 doanh nghiệp thu mua

Nếu chỉ cần:

```
10,000 user × 100k VND / tháng
= 1 tỷ / tháng
```

---

# 3. Business Model

## Subscription

### Basic

100k / tháng

Tính năng:

- thu mua
- công nợ
- CRM

### Pro

300k / tháng

- kho
- bán hàng
- báo cáo

### Enterprise

1–3 triệu / tháng

- nhiều user
- analytics
- integration
